# 🔬 Intelligent Research Layer

A full-stack AI-powered research engine that takes a user query, autonomously fetches news & web sources, filters and ranks them, runs a multi-agent debate, and synthesizes a final answer using LLMs.

**Stack:** Python (Starlette/FastAPI) · React (Vite + Tailwind) · Groq LLM · NewsAPI

---

## 🚀 Getting Started

### Prerequisites

| Tool | Version |
|------|---------|
| Python | 3.11+ |
| Node.js | 18+ |
| npm | 9+ |

---

### 1. Clone the Repository

```bash
git clone https://github.com/Rawdamm/intelligent-research-layer.git
cd intelligent-research-layer
```

---

### 2. Python Backend Setup

```bash
# Create & activate virtual environment
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Download the spaCy language model (required)
python -m spacy download en_core_web_sm
```

#### Configure API Keys

```bash
# Copy the template
cp .env.example .env
```

Then open `.env` and fill in your keys:

```env
GEMINI_API_KEY=your_gemini_api_key_here
NEWS_API_KEY=your_newsapi_key_here
GROQ_API_KEY=your_groq_api_key_here
```

Get your free keys here:
- 🔑 **Groq** → https://console.groq.com/keys
- 🔑 **NewsAPI** → https://newsapi.org/register
- 🔑 **Gemini** → https://aistudio.google.com/app/apikey

#### Start the Python Server

```bash
python app.py
# Server runs at http://localhost:8000
```

---

### 3. React Frontend Setup

```bash
cd Hackathon-Project

# Install dependencies
npm install

# Start the dev server
npm run dev
# Frontend runs at http://localhost:5173
```

Open your browser at **http://localhost:5173** 🎉

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/health` | Health check & model info |
| `POST` | `/research` | Run full intelligence pipeline |
| `POST` | `/debate` | Run multi-agent debate on a topic |
| `POST` | `/graph` | Generate knowledge graph |

### Example Request

```bash
curl -X POST http://localhost:8000/research \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the impact of AI on healthcare?"}'
```

## 🧠 Intelligence Pipeline

1. **Agent Planning** — Decomposes the query into sub-tasks
2. **Source Fetching** — Fetches relevant articles via NewsAPI + web scraping
3. **Filtering** — Keyword relevance filter removes off-topic results
4. **Scoring & Ranking** — Semantic similarity scoring ranks sources
5. **Debate Analysis** — Multi-agent LLM debate surfaces opposing viewpoints
6. **LLM Synthesis** — Final answer generated by Groq (llama-3.3-70b)
