# Intelligent Research Layer — React Frontend

A modern React (Vite + Tailwind CSS v4) frontend for the Intelligent Research Layer project.

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server (requires Python backend running on port 8000)
npm run dev
```

Open **http://localhost:5173** in your browser.

> ⚠️ The Python backend must be running first. See the [root README](../README.md) for setup instructions.

## Tech Stack

- **React 19** with Vite
- **Tailwind CSS v4**
- **Framer Motion** — animations
- **React Flow (@xyflow/react)** — knowledge graph visualization
- **React Icons**

## Build for Production

```bash
npm run build
# Output in dist/
```
